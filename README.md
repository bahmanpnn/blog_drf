# Getting Started with Create React App
1- go to https://nodejs.org/ and download lts version of node js
2- install nodejs and test it ==> node --version,npm --version
3- make frontend directory and cd directory

## Create and Use React 
1- create react app==> npx create-react-app .
2- install required dependencies==> npm install axios

### Run Project
1- python manage.py runserver
2- npm start
